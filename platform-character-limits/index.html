How to choose

Use official limits when they exist.

Some platforms publish exact maximums, while others expose behavior through product UI or editorial guidance. TextFit separates verified limits from practical writing ranges so the counter does not pretend every recommendation is an official rule.

FAQ

Useful before publishing.

Does TextFit upload my text?

No. The current version counts pasted text in your browser with local JavaScript.

What should I use for CJK-heavy copy?

Use the weighted width and bytes readings when a platform may count full-width characters, Japanese, Chinese, Korean, or emoji differently.

Can platform rules change?

Yes. Use source links for verified limits and treat writing-guide ranges as practical starting points.
